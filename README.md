# FakeAPI.io - API
This project will contain a set of APIs ready to be used in your projects while in development.

## Installation
1. Clone this project:
```bash
git clone git@github.com:FakeAPI-Project/fakeapi-api.git
```

2. Install dependencies:
```bash
cd fakeapi-api/
npm install
```

3. Run it:
```bash
npm run dev
```
